# 🚀 Zulog-AI-Trader: High-Performance C++ Trading Bot

A high-frequency, low-latency automated trading engine designed for real-time market analysis and execution on the Binance exchange.

## ✨ Features
- **Real-time Data Streaming:** Utilizes `IXWebSocket` for instantaneous market price updates.
- **High Performance:** Built with **C++** to ensure millisecond-level analysis and execution speed.
- **Secure Configuration:** Sensitive API credentials are managed through external configuration files, keeping them safe from source control.
- **Robust Logic:** Includes advanced trailing stop-loss mechanisms and real-time balance monitoring.

## 🛠 Tech Stack
- **Language:** C++ (C++17/20)
- **Networking:** IXWebSocket
- **Data Handling:** nlohmann/json
- **Build System:** CMake
